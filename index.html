I'm a Computer Technology graduate and AWS-certified cloud engineer based in Lagos, Nigeria. I build scalable, cost-efficient cloud infrastructure that solves real problems — not just theoretical solutions, but deployed systems that actually run.

I got into cloud after realising that infrastructure is the foundation everything else is built on. A great app means nothing if it can't scale, stay secure, and stay online. That's the problem I want to solve.

I'm actively building toward a remote cloud engineering role. Every project I ship is another proof point that I can operate in a real cloud environment — not just pass a certification exam.
